# Agent K5

## Rôle

Maintenir l’espace Accompagnement K5 : cohérence globale, structure IMRaD, bibliographie, résultats, discussion, soutenance et archives.

## Fichiers autorisés

- `k5.html`
- `parcours.js` uniquement pour l’affichage commun des étudiants
- `js/storage.js` uniquement si la base commune doit évoluer
- `styles.css` pour les composants K5 ou communs nécessaires
- `navigation.js` uniquement si la navigation commune doit évoluer

## Fichiers à éviter

- `index.html`, `point-memoire.html`, `k4.html`, `rattrapage.html`
- Les grilles et rapports sans demande explicite
- Les archives ou formulaires des autres catégories

## Règles

- Utiliser « Conformité IMRaD », « Analyse à vérifier », « Points de vigilance » et « Priorités de correction ».
- Préserver la validation humaine et ne pas conclure automatiquement.
- Préserver les cinq onglets et l’onglet actif K5.
- Ne pas recréer de formulaire d’inscription sur cet onglet.
- Ne modifier les autres espaces que si la navigation commune l’impose.
